# Week 2: Python Basics

This week I practised variables, data types, input, output and simple
calculations in Python.

## Files
 -variables.py- Stores and prints personal information using variables 
 -data_types.py - Shows examples of strings, integers, floats and booleans 
 -simple_receipt.py - Asks the user for an item and calculates the total cost 
 -profile_generator.py - Asks questions and prints a formatted profile summary 

## How to Run

cd week-2-python-basics
python variables.py
python data_types.py
python simple_receipt.py
python profile_generator.py
